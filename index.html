Welcome to dosemu2 home page

dosemu2 is a virtual machine that allows you to run DOS programs under linux. It is an attempt to re-incarnate the famous dosemu project under GPLv2+ license (dosemu1 license is controversial).

Get source code

You can get source code from git repository with the following command:
git clone https://github.com/stsp/dosemu2.git
or browse the repository .

Note: there are no official releases of dosemu2 yet. If you want to help with dosemu2 development or testing, for now you need to use the code from git.

A pre-release binary package for Fedora-21 x86-64 is here.
